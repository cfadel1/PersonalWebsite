Portfolio

Machine Learning

Wrote a Python script that trains a neural network with TensorFlow and applied Deep Learning to automatically recognize hand-written digits. This project was completed with the purpose of learning more about TensorFlow. For more information, check out the blog section on TensorFlow's Convolutional Neural Network.

Implemented Using: Python, TensorFlow Library

Keylogger

A Console Application capable of logging all the keyboard and mouse inputs using Windows API. The logs are then sent to the user by a generated PowerShell script. All the logs emailed are encrypted. The program has also the capability of decrypting these logs. The project was completed part of an Ethical Hacking course strictly for Educational purposes.

Implemented Using: C++, Windows API

Pedometer

An Android Application capable of guiding a user to a destination while correcting the user's wrong turns along the way. The algorithm uses a path-finding algorithm similar to Dijkstra's algorithm.

Implemented Using: Java

Stocks to Date

Dynamic information interface windows application that fetches the current price in Canadian Dollars of currencies such as Gold, US Dollar and European Euro.

Implemented Using: C#, Yahoo Finance
