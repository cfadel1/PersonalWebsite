Projects

Machine Quizzer

A Windows Application that enables a user to quiz a machine. The user will enter the name of a common person/object then upload a picture of that input and having the machine predict whether or not the picture matches the user's input. The application uses Google's Tensor Flow Library, for numerical computation using data flow graphs in order to teach the machine about the user's input.

Implemented Using: C++, TensorFlow Library

Keylogger

A Console Application capable of recording all the keyboard and mouse inputs using Windows API. It possesses encryption and decryption capabilities. The project was completed part of an Ethical Hacking course strictly for Educational purposes

Implemented Using: C++, Windows API

Pedometer

An Android Application capable of guiding a user to a destination while correcting the user’s wrong turns along the way. The algorithm uses a pth-finding algorithm similar to Dijkstra's algorithm.

Implemented Using: Java

Stocks to Date

Dynamic information interface windows application that fetches the current price in Canadian Dollars of currencies such as Gold, US Dollar and European Euro.

Implemented Using: C#, Yahoo Finance
